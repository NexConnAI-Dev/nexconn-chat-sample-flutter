# Nexconn Flutter Samples Code

This directory contains two kinds of sample assets:

- standalone code snippets for direct documentation quoting
- runnable Flutter sample apps under `apps/`

Files:

- `01_initialize_and_connect_chat_sdk.dart`
- `02_send_text_message_in_direct_channel.dart`
- `03_send_media_file_message_in_direct_channel.dart`
- `04_create_group_channel_and_send_message.dart`
- `05_get_recent_direct_and_group_channels.dart`
- `06_set_no_disturb_for_direct_and_group_channels.dart`
- `07_get_group_members.dart`

Runnable sample apps:

- `apps/sample_initialize_and_connect_chat_sdk`
- `apps/sample_send_text_message_in_direct_channel`
- `apps/sample_send_media_file_message_in_direct_channel`
- `apps/sample_create_group_channel_and_send_message`
- `apps/sample_get_recent_direct_and_group_channels`
- `apps/sample_set_no_disturb_for_direct_and_group_channels`
- `apps/sample_get_group_members`

Notes:

- These files prioritize readability for documentation examples.
- Each sample is kept as independent as possible for direct reuse.
- Replace any `YOUR_*` placeholder values with real business data.
- The `apps/` directory is the runnable version of these samples, with one small
  Flutter project per scenario.
